*Learning how to display information on the screen.*

*Learning how to store different types of data and use them later.*

- ✅ `str` (Text)
- ✅ `int` (Whole numbers)
- ✅ `float` (Decimal numbers)
- ✅ `bool` (True or False)

*Learning how to get information from the user and use it in a program.*

- ✅ `input()`
- ✅ Type Casting (`int()`, `float()`) — Changing data into the type I need.

### ➕ Math

*Learning how to perform calculations in Python.*

- ✅ Arithmetic Operators

### 🔤 Strings

*Learning how to work with text and change it.*

- ✅ String Concatenation — Joining text together.
- ✅ f-Strings
- ✅ `len()` — Checking how many characters are in text.
- ✅ String Slicing — Taking a part of a string.
- ✅ `.upper()` — Making text uppercase.
- ✅ `.lower()` — Making text lowercase.
- ✅ `.capitalize()` — Making the first letter uppercase.
- ✅ `.replace()` — Replacing part of a string.
- ✅ `.strip()` — Removing extra spaces.
